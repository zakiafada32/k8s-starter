# Basic CRUD API Kubernetes

## App info

- ### Features
  - Authentication
    - Login
    - Refresh token
  - Users
    - Get current user info
    - Get all users (admin role)
    - Get user by id (admin role)
    - Register user (admin role)
    - Update name of the user (admin role)
    - Delete user (admin role)
